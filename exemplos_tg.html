Exemplos - Torção Generalizada


February 1, 2025

Seguiremos a notacao dada na nota principal do projeto de Torção Generalizada em Grupos Nilpotentes-por-Finitos.

Example 1. Seja N = ⟨x,y,∣[x,[x,y]],[y,[x,y]]⟩ o grupo nilpotente livre de classe 2. Assim, defina G = C2 ⋉ N com xσ = x−1 e yσ = y−.

  1. Todo elemento de G é da forma xnym[x,y]kσα com n,m,k ∈ℤ e α ∈{0,1}.
  2. N′ = ⟨[x,y]⟩.
  3. N′⊆ Z(G)
  4. N′⊈T∙(G)
  5. Contra-exemplo para [G∕G′ finito implica T∙(G) = G] no caso n > 1.
  6. Contra-exemplo para [Z(G) ∩ γn(N) implica zγn+1G de ordem infinita] no caso n > 1.

Note que, N′ = ⟨[x,y]⟩⊆ Z(G).
